Built a database for a jacket store. A designed a template with multiple tables, relationships, constraints, users and diagrams. Used mysql workbench to create database, tables and EER diagram.

Created users with seperate permissions. Inserted some dummy data to test the database. Prepared a pdf document for the users to read about the database. This is a on going project as it might require further updation and addition in database.
