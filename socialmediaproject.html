Social Media Project in Python django and bootstrap

  • GitHub

A social media project created to share your projects, skills, education and connect with others. Users can also create groups and start sharing ideas and projects details with others. Users can vote for the projects, leave feedback and message each others. project is work in progress and many things will be added in future.

Process:

  • User registration and authentication with permissions.
  • Profile Management with email settings.
  • Profile view, project views and study group views.
  • User can create profile, update projects, create study groups, comments and like projects.
