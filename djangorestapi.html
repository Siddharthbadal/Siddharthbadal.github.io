Social Media Project in Python django and bootstrap

  • GitHub

ECommerce store API developed using Django rest framework. It is designed to create, read, update and delete products from a store.
API has filters applicabale on various fields of the products with proper validations on fields.
Created Error handler and exception handler functions to take care of the error pages and messages with status code over the django default error codes.
Applied JWT Authentication to access various endpoints. Email configuration for password verification.
AWS S3 Bucket is used to store products images. Swagger documantation to work with api endpoints.

Endpoints:

  • View all products
  • View one product
  • Create a product
  • Update product details
  • Delete a product
  • Register and login user
  • Update user account detailsa and Update password
