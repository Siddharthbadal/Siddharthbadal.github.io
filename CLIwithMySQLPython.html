Course registrar command line interface with python and MySQL and Typer

  • GitHub

Created a database and tables in mysql from python. Initiated an inbuilt dataset, applied triggers and operational constraints. Implemented create, read, edit, delete, and display functionalities on data. Used typer and rich library to display results in a user friendly way on command line.

Commands:

  • Add a student
  • Add a course and course prerequisite
  • Enroll and Un-enroll students
  • Set grades to mark completed
  • View top students and courses
  • Get a student transcript
