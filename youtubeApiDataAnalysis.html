YouTube API Data Fetching and Analysis

  • GitHub

Fetching data for multiple channels and analyzing Free Code Camp Youtube channel through Youtube API. Data analysis is performed using Python, Pandas, matplotlib and seaborn on Jupyter Notebook. Process included data exploration, data cleaning, finding null values, working with data types, formatting date time stamp and duration. Adding new columns, sorting values etc.

Analysis Findings:

  • Top Performing Videos in terms of Views, Likes and comments
  • Videos by months and weeks.
  • Comparision by views, likes and comments in relations.
  • Active months, Active Days, Like to views ratios
  • Multiple charts, Word Cloud
