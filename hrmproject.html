Human Resource Management System with Python Django, Bootstrap

  • GitHub

Full stack Human Resource Management System in django and postgresql with job portal functionalities. Users can register and submit their job applications. and admin can verify and update the status on the application.
A complete customized admin panel, with validations on frontend as well as on backend. Functionalities include file upload, image upload, read-only and editable admin fields. Multiple kind of fields inputs are used in user form to submit applications.

Process:

  • User registration with multiple fileds; checkboxs, dropdowns, date picker, update documents.
  • Form validation on each fileds. Eg. date validation, email validation, strings and integer validations etc
  • User Validations on fronted (for user inputs) and backend (for admin panel)
  • Forms widegts on multiple feilds to manage values, labels, and input
  • Customized admin panel with read only fields and filters for admin and support staff.
  • One to one chat messages for each candidates between admin and staff.
